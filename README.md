AcmeNewsBundle
========================

Installation
--------------
* `git clone https://github.com/robaks/AcmeNewsBundle.git ./AcmeNewsBundle`
* `cd AcmeNewsBundle`
* `composer install -n`
* `$ chmod -R 755 var/`
* `$ chmod -R 755 web/`
* Create in mysql user with database that have the same names as the user. (name - "news", pass - "admin123")
* Import "AcmeNewsBundle/dump/news.sql"
